# hash-actions

A utility for working with the `hashchange` event and `window.location.hash`.

## Development

### Install dev dependencies

```
npm install
```

### Lint

```
npm run lint
```

### Build

```
npm run build
```

### Check file sizes

Print file sizes (source, minified, gzipped, minified gzipped etc.)

```
./print-file-sizes.sh
```
